Redline Name Cleaner

Standardize comment and tracked-change names in Word documents before you share them externally. Rename internal editors, research assistants, or other review identities into a clean outward-facing label without sending the document to any server.

Free & open source 100% in-browser Multiple files supported

What It Does

  • Detects comment and revision names across one or more .docx files.
  • Lets you keep file-specific settings separate so different documents are never mixed together.
  • Makes author merges explicit before download, so no identity collapse happens by surprise.
Privacy Your documents are processed directly in your browser. No file is uploaded. No server receives your document content.

How To Use

  • Upload one file or a batch of files.
  • Open a file card to review the names Word found in comments and tracked changes.
  • Select only the people you want to unify, then apply a typed name or a saved preset.
  • Download each edited file separately, or bundle all edited files into one zip.

Important Merge Behavior

Word treats exact output matches as one visible identity. If two or more original names in the same document are changed to the exact same replacement name, Word will display those comments and revisions as coming from the same visible author identity.
